# Math CLI Calculator

Консольний застосунок для інтерпретації математичних виразів:
- Побудова бінарного дерева (TreeNode)
- Обчислення через обход дерева
- Історія обчислень у JSON

## Команди
python -m src.console run "(2+3)*4"
python -m src.console list
python -m src.console delete 0

## Тести
python -m pytest -q
